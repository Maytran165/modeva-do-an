Import snapshot vào LocalStorage

Snapshot mặc định trong repo nằm ở data/snapshot.anonymized.json. Import sẽ ghi đè các key tương ứng trong localStorage.

Export snapshot ẩn danh (từ máy bạn)

Export sẽ tự ẩn danh email/sđt/địa chỉ trong các key đơn hàng/khách hàng. Dữ liệu sản phẩm (giá/size/màu/mẫu mã) vẫn giữ nguyên để demo/chấm.
